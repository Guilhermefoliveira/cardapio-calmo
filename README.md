# Cardápio Online - Calmô Cafés e Cookies

Este projeto é um cardápio online estático, desenvolvido com React, Vite e Tailwind CSS, projetado para refletir a identidade minimalista e sofisticada da marca Calmô.

## Tecnologias

- **React**: Biblioteca para construção da interface.
- **Vite**: Ferramenta de build e servidor de desenvolvimento.
- **Tailwind CSS**: Framework de estilização utilitária.
- **Lucide React**: Ícones leves e flexíveis.

## Como Rodar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse `http://localhost:5173` no seu navegador.

## Estrutura do Projeto

- `src/components`: Componentes reutilizáveis (Hero, ProductCard, Footer).
- `src/pages`: Páginas da aplicação (Home).
- `src/assets`: Imagens e recursos estáticos.
- `tailwind.config.js`: Configuração de temas e cores personalizadas da marca.

## Cores da Marca

- **Creme**: #F5F1EB (Fundo principal)
- **Café**: #8B6F47 (Títulos e elementos de destaque)
- **Laranja Suave**: #D4A574 (Detalhes e hover)
